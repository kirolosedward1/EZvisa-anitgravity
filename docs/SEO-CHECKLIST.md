# SEO Checklist - EZvisa.net

Use this checklist to ensure every page is fully optimized for search engines.

---

## New Page Launch Checklist

### Pre-Launch (Required)
- [ ] **Title Tag** (50-60 characters, includes primary keyword)
- [ ] **Meta Description** (150-160 characters, includes CTA)
- [ ] **URL Structure** (clean, keyword-rich, lowercase, hyphens)
- [ ] **H1 Tag** (one per page, includes primary keyword)
- [ ] **H2-H6 Tags** (logical hierarchy, includes secondary keywords)
- [ ] **Content Length** (minimum 800 words for main pages, 1,200+ for country pages)
- [ ] **Keyword Density** (1-2% for primary keyword, natural placement)
- [ ] **Internal Links** (3-5 relevant links to other pages)
- [ ] **External Links** (1-2 to authoritative sources)
- [ ] **Images Optimized** (compressed, WebP format, proper dimensions)
- [ ] **Alt Text** (descriptive, includes keywords naturally)
- [ ] **Schema Markup** (appropriate structured data implemented)
- [ ] **Canonical Tag** (points to correct URL)
- [ ] **Mobile Responsive** (tested on multiple devices)
- [ ] **Page Speed** (Lighthouse score 90+)

### Post-Launch (Within 7 Days)
- [ ] Submit URL to Google Search Console
- [ ] Check indexing status
- [ ] Monitor Core Web Vitals
- [ ] Verify schema markup (Google Rich Results Test)
- [ ] Check for crawl errors
- [ ] Add to sitemap (auto-generated)
- [ ] Create internal links from existing pages
- [ ] Share on social media

## Blog Post SEO Checklist

### Content Creation
- [ ] Target keyword identified (search volume 100+)
- [ ] Secondary keywords researched (3-5 LSI keywords)
- [ ] Competitor content analyzed
- [ ] Content outline created
- [ ] Minimum 1,200 words written
- [ ] Engaging introduction (addresses search intent)
- [ ] Clear subheadings with keywords
- [ ] Conclusion with CTA
- [ ] FAQ section included (3-5 questions)

### On-Page Optimization
- [ ] Title tag optimized (60 characters max)
- [ ] Meta description optimized (160 characters max)
- [ ] URL slug optimized (short, descriptive)
- [ ] H1 includes primary keyword
- [ ] H2s include secondary keywords
- [ ] First paragraph includes primary keyword
- [ ] Keywords used naturally throughout (1-2% density)
- [ ] Bold/italics for emphasis
- [ ] Bullet points for readability
- [ ] Short paragraphs (3-4 sentences max)

### Media & Formatting
- [ ] Featured image added (1200x630px)
- [ ] Alt text for featured image
- [ ] 3-5 supporting images
- [ ] All images compressed (<100KB each)
- [ ] Images have descriptive filenames
- [ ] Videos embedded (if applicable)
- [ ] Infographics included (if relevant)

### Links & Schema
- [ ] 5+ internal links to relevant pages
- [ ] 2-3 external links to authoritative sources
- [ ] All links open in new tab (external)
- [ ] Article schema markup added
- [ ] Breadcrumb schema added
- [ ] FAQ schema added (if FAQ section exists)

### Pre-Publish Review
- [ ] Spell check and grammar check
- [ ] Read aloud for flow
- [ ] Check all links work
- [ ] Preview on mobile device
- [ ] Yoast/RankMath SEO score: Green
- [ ] Readability score: Good

### Post-Publish
- [ ] Share on all social media platforms
- [ ] Submit URL to Google Search Console
- [ ] Add to email newsletter
- [ ] Create pin for Pinterest (if applicable)
- [ ] Monitor comments and engagement
- [ ] Update internal linking structure

## Country Page SEO Checklist

### Content Requirements
- [ ] Country name in title tag
- [ ] "Schengen visa" or "tourist visa" in title
- [ ] Year included in title (e.g., "2024")
- [ ] H1: "[Country] Schengen Visa Requirements from UAE"
- [ ] Introduction paragraph (100-150 words)
- [ ] Overview section
- [ ] Required documents section (detailed list)
- [ ] Application process steps (numbered list)
- [ ] Processing time information
- [ ] Common rejection reasons (5-7 points)
- [ ] Tips for successful application (5-7 points)
- [ ] FAQ section (5-7 country-specific questions)
- [ ] Minimum 1,200 words total

### Visual Elements
- [ ] Country flag image
- [ ] Sample visa image (if available)
- [ ] Document checklist infographic
- [ ] Map showing country location
- [ ] Embassy/consulate photo
- [ ] All images have alt text with country name

### Links & Navigation
- [ ] Link to main documents page
- [ ] Link to 2-3 neighboring countries
- [ ] Link to related blog posts
- [ ] Link to application page (CTA)
- [ ] Breadcrumb navigation
- [ ] Previous/Next country navigation

### Schema Markup
- [ ] HowTo schema for application process
- [ ] FAQ schema for Q&A section
- [ ] Breadcrumb schema
- [ ] Organization schema (inherited)

### Uniqueness Check
- [ ] Content is unique (not copied from other countries)
- [ ] Country-specific information included
- [ ] Embassy details specific to UAE
- [ ] Processing time specific to this country
- [ ] Fee information specific to this country

- [ ] Identify secondary keywords (5-10)

- [ ] Analyze top 10 ranking pages
- [ ] Identify content gaps
- [ ] Check for featured snippet opportunities
- [ ] Review "People Also Ask" questions
- [ ] Check related searches at bottom of SERP

### Keyword Tracking Setup
- [ ] Add to ranking tracker
- [ ] Set benchmark position
- [ ] Set target position goal
- [ ] Set tracking frequency (weekly)
- [ ] Note current search volume
- [ ] Note keyword difficulty score

## Link Building Checklist

### Before Outreach
- [ ] Create linkable asset (content worth linking to)
- [ ] Identify target websites (20-50)
- [ ] Find contact information (email addresses)
- [ ] Check domain authority (aim for 30+)
- [ ] Verify site is relevant to your niche
- [ ] Ensure site has organic traffic
- [ ] Check for spam signals
- [ ] Personalize outreach message

### Outreach Email Must Include
- [ ] Personalized greeting (use recipient's name)
- [ ] Why you're reaching out (specific reason)
- [ ] Value proposition (what's in it for them)
- [ ] Clear call-to-action
- [ ] Your contact information
- [ ] Professional signature

### After Link Acquired
- [ ] Document link details (URL, anchor text, date)
- [ ] Monitor link status monthly
- [ ] Check if link is followed/nofollowed
- [ ] Track traffic from referring domain
- [ ] Build relationship with site owner
- [ ] Consider future collaboration

## Emergency SEO Fixes

### If Rankings Drop >5 Positions
1. [ ] Check Google Search Console for issues
2. [ ] Review recent algorithm updates
3. [ ] Audit recent changes to the page
4. [ ] Check competitors' changes
5. [ ] Verify technical issues (404s, slow speed)
6. [ ] Review backlink profile for toxic links
7. [ ] Check for duplicate content issues
8. [ ] Implement fixes immediately
9. [ ] Monitor recovery over 2-4 weeks

### If Traffic Drops >20%
1. [ ] Verify tracking is working correctly
2. [ ] Check if site is indexed
3. [ ] Review Google Search Console for penalties
4. [ ] Check server uptime and speed
5. [ ] Analyze competitors for new content
6. [ ] Review seasonal trends (is drop expected?)
7. [ ] Check for major SERP changes
8. [ ] Review and refresh old content
